# Chelstone Houses Network

## Project Overview
Chelstone Houses Network is a static HTML website developed to advertise and sell houses located in **Chelstone, Lusaka (Zambia)**.  
The website uses a **social-network-style layout** where sellers can post house listings and buyers can view properties, interact through likes, comments, and messages, and follow community housing updates.

## Website Structure
The website is built using **HTML only** and structured with tables to organize content into:
- Header section
- Navigation menu
- Main house listings feed
- Community interaction panel
- Footer section

Each house listing includes:
- Seller name
- House description
- Real house images
- Price in Zambian Kwacha
- Interaction links (Like, Comment, Message)
